This is a column.

To make a column, give an element the class column.

<div class="column"> </div>

To be more specific, this is a one column element.

<div class="one column"> </div>

This is a red column.

<div class="red column"> </div>

As you can see, it's larger than the yellow one.

In fact, it's double the size: a two columns element.

<div class="two columns red"> </div>

There can be up to 6 columns in a container.

So, 1 yellow, 2 red ...

1 + 2 + ... = 6

exactly, this is a three columns element.

<div class="three columns blue"> </div>

Classes are very useful indeed
